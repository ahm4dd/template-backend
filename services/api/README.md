# API Service

Main HTTP API service for the template.

## Environment

Required variables:

- `NODE_ENV`
- `PORT`
- `DATABASE_URL`

Local overrides supported:

- `.env.local`
- `.env.development.local`
- `.env.production.local`

## Scripts

- `pnpm dev`
- `pnpm start`
- `pnpm lint`
- `pnpm typecheck`
